Dock Scheduling System

เปลี่ยนการบริหาร Dock จากกระดาษและไลน์แจ้งเตือน ให้เป็นระบบเรียลไทม์ที่มองเห็นทุกอย่างในที่เดียว — จัดคิวรถเข้า-ออก ตรวจจับการชนกันของตารางอัตโนมัติก่อนเกิดปัญหาจริงหน้างาน

  • Dashboard ภาพรวมการทำงาน — KPI จำนวน trip วันนี้ On-time % และโหลดเฉลี่ย คำนวณอัตโนมัติ
  • Dock Map เรียลไทม์ มองเห็นสถานะ 18 Dock พร้อมกัน พร้อม Conflict Detection อัตโนมัติ
  • Swimlane Planner — จัดคิวงาน Dock เทียบเวลา แจ้งเตือนการชนกันก่อนเกิดปัญหา
  • Trip List & WIP Tracker — ติดตามทุก trip พร้อม Delay Alert อัตโนมัติและ Export Excel

Daily Operation Dashboard

ระบบรายงานเหตุการณ์เสี่ยง (Near-Miss)

ให้พนักงานหน้างานรายงานเหตุการณ์เสี่ยงได้ทันทีผ่าน QR code ที่ติดตามจุดเสี่ยงในโกดัง รองรับการแจ้งแบบไม่ระบุชื่อเพื่อลดความกังวลในการรายงาน

  • สแกน QR ที่ป้ายเตือนหน้างาน เชื่อมตรงไปยังฟอร์มรายงานจุดนั้น
  • แนบรูปได้สูงสุด 3 รูป เก็บผ่าน Google Drive อัตโนมัติ
  • ระบบยืนยันตัวตนด้วย API key ป้องกันการยิงข้อมูลเข้าระบบโดยไม่ผ่านแอป
  • บีบอัดรูปฝั่งผู้ใช้ก่อนอัปโหลด ลดเวลาโหลดบนมือถือ
